Commercial use of Jd requires a Jd Primary License.

Non-commerical use of Jd is free and does not require a Jd Primary License. This encourages use of Jd and will help build the talent pool of J programmers familiar with Jd. Their active participation in the J forums will provide depth and experience in free Jd support.

License fees are in USD payable in advance.

Jd Primary License

A Jd Primary License is an annual fee. You must have a current Jd Primary License to run Jd commercially.

Jd Primary License is $20,000 each year and in the first year includes 2 Jd Install Licenses.

Jd Install License

A Jd Install License is a one time fee that allows you to install, maintain, and run Jd on a machine. You can install multiple copies of Jd on the machine and it can run in multiple instances and on multiple cores.

An exisiting Jd Install License can be used to install on a different machine only if use on the previous machine is stopped.

A Jd Install License is a $4000 one time fee.

Jd Special License

Jsoftware offers a Jd Special License to encourage use of Jd where there is mutual benefit. A company can provide justification and request a Jd Special License.

Jd Special License is $1,000 each year and in the first year includes 10 Jd Install Licenses.

In response we will send an invoice and you can start commercial use of Jd as soon as we receive payment.
