intx

intx table col new-type

converts col from its current type to new-type

col current type must be int/int1/int2/int4

new-type must be int/int1/int2/int4/intx

new-type intx converts col to type that takes least space without losing data

NB. example intx
jdadminx'example'[olddan=:jdaccess''
jd'gen test f 5'
jd'info schema'
jd'intx f int intx'
jd'info schema'
jdaccess olddan

key

key [/in] table pairs

Returns last table row that matches each of the rows in pairs (_1 for not found).

With /in option, returns table rows that match any row in pairs.

See key for info.

NB. example key
jdadminx'example'[olddan=:jdaccess''
jd'gen test f 5'
jd'key f';'int';101 103
jd'key f';'int';103 101 200
jd'key f';'int';103 101;'byte';'DX'
jd'key f';'boolean';0 1
jd'key /in f';'boolean';1
jdaccess olddan
