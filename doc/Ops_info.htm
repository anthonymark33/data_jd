info

info type [...] - info about current DB

agg                         - aggregators
last                        - cmd time space parts
schema      [table[column]] - normal cols (non jd...)
summary     [table]         - active,deleted rows
table       [table]
validate    [table[column]]
validatebad [table[column]]
varbyte     [table[column]]
ref         [table[column]] - ref cols

Tables are sorted and cols are in default selection order

Schema and similar types treat ptable as a single table
validate and validatebad list ptable parts

summary     - tables active deleted
summary f^* - tables that make up the ptable

table       - tables
table f^*   - tables that make up the ptable

NB. example info
jdadminx'example'[olddan=:jdaccess''
jd'gen ref2 a 3 0 b 2'
jd'info table'
jd'info summary'
jd'info schema'
jd'info schema a'
jd'info schema a akey'
jdaccess olddan
