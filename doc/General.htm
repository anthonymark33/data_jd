Install

Install Jd (addon data/jd) with JAL/Package Manager. Jd requires J64 (805 or later) on Windows, Linux, or Mac.

Run

   load'jd'
   jdwelcome_jd_ NB. run this sentence for important information

The welcome suggests how to get started. In particular how to list and run tutorials and examples.

If you don't have a valid key you see a message to that effect.
