Install

Install Jd (addon data/jd) with JAL/Package Manager. Jd requires J64 (805 or later) on Windows, Linux, macOS, and RPI.

Run

   load'jd'
   jdwelcome NB. run this sentence for important information

The welcome suggests how to get started. In particular how to run tutorials.

If you don't have a valid key for a license you see a message to that effect.
