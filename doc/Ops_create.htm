createcol

createcol table col type
createcol table col byte [trailing-shape]
createcol table col type data
createcol table col byte [trailing-shape] data

creates new col with default values or data

NB. example createcol
jdadminx'example'[olddan=:jdaccess''
jd'createtable f'
jd'createcol f a int'
jd'insert f a';12 13 14
jd'createcol f b int'
jd'createcol f d int';88 89 90
jd'reads from f'
jdaccess olddan

createtable

createtable [/a ...] table [coldefs]
createtable /pairs [/types] [/a ...] table ; pairs

/a min_rows multiplier extra_rows
when insert requires mapped col file resize, new size is:
min_rows>.>.multiplier*extra_rows+current_rows+new_rows
default is 2000 1.5 0

/pairs

/types - pairs have types in parens - cola(edate)

createtable with 0 or more cols

coldefs - col type [trailing-shape]
string arg coldefs separated by , or LF
boxed arg coldefs in separate boxes

NB. example createtable
jdadminx'example'[olddan=:jdaccess''
jd'createtable f'
jd'createtable g a int , b byte 4'
jd'createtable h';'a int';'b byte 2'
jd'createtable i a int',LF,'b byte 7'
jd'info schema'
jd'createtable /pairs j';'a';2 3 4;'b';3 4$'abcdefghijkl'
jd'reads from j'
jdaccess olddan
