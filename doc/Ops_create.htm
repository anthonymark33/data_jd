createdcol

createdcol table col type verb
createdcol table col byte [trailing-shape] verb
creates a derived col
derived col data is not from a mapped file
derived col data is the result of verb derived_verb defined in custom.ijs

a derived byte 3 col could be derived from the category part of a byte 20 col
a derived int col could be derived from the year in an edatetime col
